Desire2Download

Source Available - Downloads content from Desire2Learn used at the University of Waterloo

What is it?

It’s a tool to download content automagically from the University of Waterloo’s LMS that professors post lecture notes and other content.

I developed it so I wouldn’t need to continuously check the Waterloo site to see if a Prof has posted new content. The script automatically downloads new content when available. I have the script download to a shared folder on DropBox so that I can access the latest files on my mobile devices or any computer.

It saves me a ton of time.

When

In Fall 2011, when the new Desire2Learn system was used in 2 of my courses.

I was inspired by Jamie Wong’s fabulous UWAngel-CLI written in Ruby.

Technology

  • Python
  • mechanize
  • The Amazing BeautifulSoup

Success?

So far Desire2Download has worked very well for me and several friends.

Check it out on github: Desire2Download
